### 기능 리스트
- HTTPS 지원
- 소셜 로그인(카카오, 애플)
- JWT 기반의 토큰을 활용한 인증
- 개인 운동 기록 생성 및 조회
- 친구 신청 및 응답
- 챌린지 생성 및 초대
- 랭킹 조회
